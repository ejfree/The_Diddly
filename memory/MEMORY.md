# The Diddly — Campaign Memory

## System
- **PF2e (Pathfinder 2e)** — always use PF2e rules, class names, and archetype terminology
- Class combinations written as "main class with X dedication" (e.g., "Cleric with Rogue dedication"), NOT "Cleric/Rogue hybrid"

## Campaign Structure
- **Arc 2.5:** Bottom 3 layers of the Pit (Seven Dooms of Sandpoint module) — trail of giants discovered at bottom
- **Arc 3 (Gap Year):**
  - G1 (Hill Giants) → G2 (Frost Giants) → G3 (Fire Giants)
  - D1 (troglodytes / Underdark entry) → D2 (kuo-toa warrens)
  - D3: Xilthraen (drow war city — D3: Vault of the Drow layout, re-skinned)

## Key Canon Rules
- **Dragons do NOT know about the Pale Choir (insanity plague).** They only perceive the drow uprising as the external threat. Their degraded judgment from the plague feels like "clarity" to them.
- **Lolth's metaplot:** Drive draconic Council insane via Xoriat + free Tiamat (imprisoned in Argonessen). Drow surface war is distraction/culling.
- **The giant chain** (G1→G2→G3→D1→D2→D3) is Lolth's design — the PCs completing it fulfills the Black Coil prophecy sequence.
- **The "flight from the university"** prophesied as the final confirmation of the Fourth Age — faculty don't know they're inside the prophecy.
